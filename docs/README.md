# 문서 작업 공간

이 폴더는 책 원고를 관리합니다.

## 파일 구성

- `SUMMARY.md`: 전체 목차
- `chapters/`: 챕터 본문
- `notes/`: 집필 메모

## 작성 규칙(초안)

- 챕터 파일은 `NN-title.md` 형식 사용
- 예제 코드는 `examples/`에 두고, 문서에서는 경로만 참조
- 챕터 집필 전 반드시 아래 기준 파일 3개를 먼저 확인
  - `docs/chapters/00-goals-and-assumptions.md`
  - `README.md`
  - `docs/SUMMARY.md`
